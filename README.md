# 项目

基于`React v18.2.0`、`antd v4.16.9`进行开发，整个页面结构由**菜单栏**、**组件区**、**画布区**、**属性区**构成；项目内则采用配置项的灵活方式，使得用户可以在每个组件区域所管辖的范围内做出多种不同的操作，且各区域中拥有一个较为完善的处理渲染组件的方式

> 截止到目前，支持的antdUI组件相对较少，后续也会继续增添与维护
